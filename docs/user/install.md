# Installation

## Requirements

- Windows 11 (Windows 10 may work but is not tested)
- Python 3.13+

## Install via uv (recommended)

```bash
uv tool install press
```

After installation, `press` is available as a standalone command without activating any virtual environment.

## Install via pip

```bash
pip install press
```

## Verify installation

```bash
press --version
```

## Uninstall

```bash
uv tool uninstall press
```
